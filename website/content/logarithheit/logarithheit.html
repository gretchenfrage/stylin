Proposed temperature system: logarithheit

Nobody asked for this

Take these constants:

  • let Z = -459.67
    (absolute zero in fahrenheit)
  • let A = \frac{100}{\ln ((100-Z)/(-Z))}
  • let B = -A \ln (-Z)
  • let C = \sqrt[A]{e}

Then, to convert from fahrenheit f to logarithheit s (for "scaled"):

s(f) = \log_C (f - Z) + B

And back again:

f\left(s\right)=C^{s-B}+Z

Cool properties:

  • 0 degrees is the same in fahrenheit and logarithheit.
  • 100 degress is the same in fahrenheit and logarithheit.
  • Temperature ranges between 0 and 100 are very close between fahrenheit and logarithheit.
  • Absolute zero is now - \infinity.
    (you would have to cool it infinitely to achieve that).
    • Implication: The domain of logarithheit is now all real numbers (- \infinity, \infinity)
  • The plank temperature is now 34,794.16 degrees logarithheit. (you can reasonably write it without scientific notation)
  • Human senses are logarithmic, temperature is a sense, logarittheit is a logarithmic scale, like decibels for sound. It makes sense.

Here's the neatest thing though:

It's easy to calculate the percent change in kelvin temperature, which is useful for physics. Let's say you're going from 25 to 75 degrees logarithheit. The fraction change in kelvin temperature equals:

C^{(75-25)} = 1.1031

To verify:

25 logarithheit = 23.1900 fahrenheit = 268.2533 kelvin

75 logarittheit = 73.1260 fahrenheit = 295.9975 kelvin

\frac{295.9975}{268.2533}=1.1031
